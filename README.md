# 🛒 Sistema de PDV (Ponto de Venda)

Este é um sistema de **Ponto de Venda (PDV)** completo e robusto, desenvolvido em **C#** para ambiente Desktop (**Windows Forms / WPF**). O projeto foi projetado para gerenciar frentes de caixa de forma ágil, integrada e segura, atendendo a todas as rotinas operacionais e fiscais de um estabelecimento comercial.

---

## 🚀 Funcionalidades Principais

O sistema cobre o fluxo de ponta a ponta de uma operação de varejo:

* **Fluxo de Caixa Completo:** Controle rigoroso de abertura de caixa, sangria (retirada de valores), suprimento (aporte de troco) e fechamento de turno.
* **Vendas Automatizadas:** Interface rápida para PDV com suporte a múltiplas formas de pagamento simultâneas (Dinheiro, Cartão de Crédito/Débito e PIX).
* **Gestão de Estoque:** Cadastro e controle automatizado de produtos, com baixa em tempo real a cada item vendido.
* **Integrações de Hardware:** Suporte a leitores de código de barras e balanças computacionais para pesagem no caixa.
* **Módulo Fiscal:** Preparado para a emissão de Notas Fiscais de Consumidor Eletrônicas (**NFC-e / SAT**).

---

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** C# (.NET)
* **Interface Gráfica:** Windows Forms / WPF (Desktop)
* **Banco de Dados:** MySQL (Persistência relacional robusta e escalável)
* **Acesso a Dados:** Entity Framework Core / Dapper *(Ajustar conforme o seu uso)*
